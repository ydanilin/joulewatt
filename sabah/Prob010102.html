Problem 1.1.2.

Problem text:

A diode has iD = 1 A at vD = 0.7 V and iD = 2 A at vD = 0.73 V. Determine IS and η.
What is rD in both cases?

Solution:

The solution is as easy as to create two equations with two unknowns. But if you attemp to do it with the complete Main Diode Formula
\[{i_D} = {I_S}\left( {{e^{{\raise0.7ex\hbox{${{v_D}}$} \!\mathord{\left/ {\vphantom {{{v_D}} {\eta {V_T}}}}\right.} \!\lower0.7ex\hbox{${\eta {V_T}}$}}}} - 1} \right)\]
you will end up fairly frustraited to do it analytically (you can try it if you do not believe:-) ).
I remember once our physics mentor told us: «The art of the science is what to neglect». The currents in the problem text are quite high for our exponent formula, therefore we can easily drop off the «1» in our equation! With this our life becomes much better.
First, let us calculate vD ∕ VT for both cases:
vD1 ∕ VT = 0.7 ∕ 0.026 = 26.9231 and vD2 ∕ VT = 0.73 ∕ 0.026 = 28.0769
Based on simplified diode current formula
\[{i_D} = {I_S} {{e^{{\raise0.7ex\hbox{${{v_D}}$} \!\mathord{\left/ {\vphantom {{{v_D}} {\eta {V_T}}}}\right.} \!\lower0.7ex\hbox{${\eta {V_T}}$}}}}} \]
we create two equations
\[ 1 = {I_S} {{e^{{\raise0.7ex\hbox{${26.9231}$} \!\mathord{\left/ {\vphantom {{26} \eta}}\right.} \!\lower0.7ex\hbox{$\eta$}}}}} \]
\[ 2 = {I_S} {{e^{{\raise0.7ex\hbox{${28.0769}$} \!\mathord{\left/ {\vphantom {{26} \eta}}\right.} \!\lower0.7ex\hbox{$\eta$}}}}} \]
Substituting IS from the above yelds
\[ 2 = \frac { e^{\frac {\hbox{$28.0769$}} {\hbox{$\eta$}} } } { e^{\frac {\hbox{$26.9231$}} {\hbox{$\eta$}} } } \]
\[ \ln 2 = \ln{ \left( \frac {28.0769} {\eta} \right) } - \ln{ \left( \frac {26.9231} {\eta} \right) } \]
from which
\[ \eta = \frac {28.0769 - 26.9231} {\ln 2} = 1.6646 \]
So, the slogan is: Use simplified diode current formula whenever possible!

Now let's see how to calculate incremental diode resistance.
According to Ohm:
\[ dr = \frac {dv}{di} = \frac {1}{\raise0.7ex \hbox{$di$} \! / \! \lower0.7ex \hbox{$dv$}} \]
Let's find the derivative of the Main Diode Formula (let's temporary name exponential part as \(\varepsilon(v)\)):
