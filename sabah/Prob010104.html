Problem 1.1.4.

Problem text:

Consider Equation
\[{i_D} = {I_S}\left( {{e^{{\raise0.7ex\hbox{${{v_D}}$} \!\mathord{\left/ {\vphantom {{{v_D}} {\eta {V_T}}}}\right.} \!\lower0.7ex\hbox{${\eta {V_T}}$}}}} + 1} \right)\]
with η = 1. If iD = 1 mA, determine IS for a vD of: (a) 0.3 V; (b) 0.5 V; (c) 0.7 V.

Solution:

Although the solution is quite simple, with this problem we would like to show how IS value affects the diode I-V curve.
The values are the following:
vD, V iD, A IS, A
0.3 0.001 9.75×10-9
0.5 0.001 4.45×10-12
0.7 0.001 2.03×10-15
Now let's graph all three cases. You see that
the greater the saturation current the earlier I-V curve starts its exponential growing (the lesser the offset voltage)
