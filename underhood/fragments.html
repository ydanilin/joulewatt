How to include HTML fragments elsewhere without CMS

Solution using jQuery

Suppose you want to include the same navigation bar or footer on several webpages. Usual solution is to use CMS like Django with their template engine. What if we need simpler solution?
The simple fragment below shows how to embed another html file using jQuery
          
<html>
  <head>
    <script>
      $(function(){
        $("#includedContent").load("b.html");
      });
    </script>
  </head>
  <body>
    <div id="includedContent"></div>
  </body>
</html>

b.html:
<p> This is my include file with HTML fragments </p>
Important!
This approach does not work when viewing html files locally due to Cross origin requests are only supported for protocol schemes: http, data, chrome, chrome-extension, https.
What we need is to run simple local webserver and good solution is in Python:
  • 1. Start cmd (good choice for Windows is conemu64) and cd to the folder above project root folder.
  • 2. Type python -m SimpleHTTPServer for Python 2 or python -m http.server for Python 3. The server listens to localhost port 8000.
  • 3. In browser, go to http://localhost:8000/joulewatt
    If your main file is index.html, it loads by default.
Provide absolute paths like /joulewatt/folder/folder/file.html in links.
